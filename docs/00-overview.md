## 🗓️ 学習ロードマップ

### Phase 1: 基礎固め（Week 1-2）
**目標**: C++テンプレートの身体化
- vector/map/setの完全理解
- 基本的なアルゴリズムの実装
- Binary Search/DFS/BFSの型を覚える

### Phase 2: パターン習得（Week 3-6）
**目標**: NeetCode 25を完全制覇
- Two Pointers
- Sliding Window
- Stack/Queue
- Binary Search応用

### Phase 3: 実戦レベル（Week 7-10）
**目標**: NeetCode 75で面接頻出問題を網羅
- Graph（BFS/DFS/Union-Find）
- Tree（全パターン）
- Backtracking
- Heap/Priority Queue
- DP基礎

### Phase 4: 仕上げ（Week 11-12）
**目標**: Mock Interviewで実戦感覚を養う
- LeetCode Mock Interview（週2回）
- 弱点パターンの集中復習
- 時間配分の最適化

## 💪 成功のための心構え

### やるべきこと ✅
- **反復練習**: 同じ問題を3回解く
- **パターン暗記**: 解法より「型」を覚える
- **毎日継続**: 短時間でも毎日触れる
- **記録**: progress-tracker.mdに必ず記録

### やってはいけないこと ❌
- 完璧主義（70%で次に進む）
- 1問に3時間粘る
- 解説を見ずに我流で解く
- 復習をしない
