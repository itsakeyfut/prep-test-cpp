# Insertion Sort - 挿入ソート

## 概要
挿入ソートは、トランプの手札を整理するように、一つずつ正しい位置に挿入していくソートアルゴリズムです。

## アルゴリズムの動き
左側を「ソート済み領域」、右側を「未ソート領域」として、未ソート領域から要素を取り出してソート済み領域の正しい位置に挿入します。

```
[3, 22, 19, 5] の場合

初期: [3] | 22, 19, 5        ← 最初の要素はソート済みとみなす

ステップ1: 22を挿入
          [3, 22] | 19, 5     ← 22は3より大きいのでそのまま

ステップ2: 19を挿入
          19を一時保存
          22を右へシフト
          [3, 19, 22] | 5     ← 19を正しい位置に挿入

ステップ3: 5を挿入
          5を一時保存
          22, 19, 3を順に右へシフト
          [3, 5, 19, 22]      ← 5を先頭に挿入
```

## コードの解説

### 主要な処理（main.cpp:12-23）
```cpp
for (size_t i{1}; i < size; ++i)
{
    auto tmp = nums[i];              // 挿入する要素を保存
    size_t j = i;

    while (j > 0 && nums[j - 1] > tmp)  // 挿入位置を探す
    {
        nums[j] = nums[j - 1];          // 要素を右にシフト
        j -= 1;
    }
    nums[j] = tmp;                   // 正しい位置に挿入
}
```

- `i`: 現在挿入しようとしている要素のインデックス
- `tmp`: 挿入する要素の値を一時保存
- `j`: 挿入位置を見つけるためのポインタ
- `while`ループ: `tmp`より大きい要素を右にシフトしながら挿入位置を探す

## 計算量
- **時間計算量**:
  - 最良: O(n)（既にソート済みの場合）
  - 平均: O(n²)
  - 最悪: O(n²)（逆順の場合）
- **空間計算量**: O(1)（追加のメモリ不要）

## 特徴
- **安定ソート**: 同じ値の順序が保たれる
- **適応的**: データが既にある程度ソート済みなら高速
- **オンライン**: データが順次到着する場合に有効
- **少ないデータに効率的**: 小規模データや部分的にソート済みのデータに向いている

## 使いどころ
- 小さいデータセット（数十要素程度）
- ほぼソート済みのデータ
- データが逐次的に到着する場合
- クイックソートやマージソートの内部処理（小さい部分配列に対して）
