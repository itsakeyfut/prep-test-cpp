# Bubble Sort - バブルソート

## 概要
バブルソートは最も基本的なソートアルゴリズムの一つです。隣り合う要素を比較して、順序が逆なら交換する操作を繰り返します。

## アルゴリズムの動き
大きい値が泡（bubble）のように後ろに浮き上がっていくイメージです。

```
[3, 22, 19, 5] の場合

パス1: 3と22比較 → そのまま
      22と19比較 → [3, 19, 22, 5]
      22と5比較  → [3, 19, 5, 22] ← 最大値が最後に確定

パス2: 3と19比較 → そのまま
      19と5比較  → [3, 5, 19, 22] ← 2番目に大きい値が確定

パス3: 3と5比較  → そのまま
      [3, 5, 19, 22] ← 完了
```

## コードの解説

### 主要な処理（main.cpp:11-20）
```cpp
for (size_t i{0}; i < size; ++i)
{
    for (size_t j{0}; j < size - 1 - i; ++j)
    {
        if (nums[j] > nums[j + 1])
        {
            swap(nums[j], nums[j + 1]);
        }
    }
}
```

- 外側のループ（`i`）: パスの回数（最大 n 回）
- 内側のループ（`j`）: 各パスでの比較回数
- `size - 1 - i`: 既に確定した要素は比較不要なので範囲を縮める
- 隣り合う要素を比較し、大きい方を後ろに移動

## 計算量
- **時間計算量**:
  - 最良: O(n²)（既にソート済みでも全比較）
  - 平均: O(n²)
  - 最悪: O(n²)
- **空間計算量**: O(1)（追加のメモリ不要）

## 特徴
- **安定ソート**: 同じ値の順序が保たれる
- **シンプル**: 理解しやすい
- **遅い**: 大量のデータには不向き

## 使いどころ
- アルゴリズムの学習用
- 非常に小さいデータ（10要素未満）
- 実用には向かない（より高速なアルゴリズムを使うべき）
