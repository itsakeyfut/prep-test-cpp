# Bogo Sort - ボゴソート

## 概要
ボゴソートは、配列をランダムにシャッフルして、ソートされているかチェックするアルゴリズムです。最も非効率的なソートアルゴリズムの一つで、実用性はありませんが、アルゴリズムの学習には面白い例です。

## アルゴリズムの動き
```
[3, 1, 2] をソートする場合

1回目: シャッフル → [2, 3, 1] → ソートされていない
2回目: シャッフル → [1, 3, 2] → ソートされていない
3回目: シャッフル → [3, 2, 1] → ソートされていない
...
N回目: シャッフル → [1, 2, 3] → ソート完了！
```

運が良ければ1回で終わりますが、最悪の場合は永遠に終わりません。

## コードの解説

### ソート済みチェック（main.cpp:7-15）
```cpp
bool isSorted(const std::vector<int> &arr)
{
    for (size_t i{1}; i < arr.size(); ++i)
    {
        if (arr[i - 1] > arr[i])
            return false;  // 逆順があればソートされていない
    }
    return true;
}
```
- 配列を先頭から順にチェック
- 1つでも逆順があればfalse

### シャッフル（main.cpp:17-22）
```cpp
void shuffle(std::vector<int> &arr)
{
    static std::random_device rd;      // 乱数生成器の初期化
    static std::mt19937 g(rd());       // メルセンヌ・ツイスタ
    std::shuffle(arr.begin(), arr.end(), g);  // ランダムに並び替え
}
```
- `std::shuffle`で配列をランダムに並び替え
- `static`で乱数生成器を1回だけ初期化

### ボゴソート本体（main.cpp:24-34）
```cpp
void bogoSort(std::vector<int> &arr)
{
    int32_t attempts{0};

    while (!isSorted(arr))  // ソートされるまでループ
    {
        shuffle(arr);       // シャッフル
        attempts++;
    }
    std::cout << "Sorted after " << attempts << " shuffles." << std::endl;
}
```
- ソートされるまで無限にシャッフルを繰り返す
- 試行回数をカウントして表示

## 計算量
- **時間計算量**:
  - 平均: O((n+1)!)（階乗時間）
  - 最良: O(n)（1回目で完了）
  - 最悪: 無限大（理論上は終わらない可能性がある）
- **空間計算量**: O(1)

## 特徴
- **非決定的**: 実行のたびに試行回数が変わる
- **極めて遅い**: 要素数が増えると現実的な時間では終わらない
- **シンプル**: コードは非常に単純

## 使いどころ
- **実用性なし**: 実際のプログラムでは絶対に使わない
- アルゴリズムの学習（悪い例として）
- ジョークや面白プログラムの実装

## 参考データ
5要素のソート: 平均120回のシャッフル
10要素のソート: 平均3,628,800回のシャッフル（現実的でない）
